# CaptureView

> Turn your capture card into a monitor.

CaptureView is an open-source, ultra-low latency viewer designed exclusively for gaming through capture cards.

No streaming.

No recording.

No scenes.

Just play.

---

## Why?

Modern capture cards are powerful enough to be used as gaming displays.

Unfortunately, today's software is designed for content creators, not gamers.

CaptureView changes that.

Plug.

Play.

Done.

---

## Philosophy

If the user has to read a tutorial...

we failed.

---

## Goals

- Ultra-low latency
- Zero configuration
- Automatic device detection
- Fullscreen
- Multi-monitor
- HDR ready
- Gaming-first experience
